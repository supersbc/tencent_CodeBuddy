# 🇨🇳 AIdatabase 信创国产化模式使用指南

## 📋 什么是信创?

**信创**(信息技术应用创新)是国家推动的IT产业自主可控战略,要求在关键领域使用国产化软硬件:

- 🖥️ **服务器**: 浪潮、华为、联想等国产品牌
- 💻 **CPU芯片**: 鲲鹏920、海光、飞腾、龙芯等国产芯片
- 🌐 **网络设备**: 华为、H3C等国产设备
- 💾 **存储**: 长江存储、宏杉等国产存储
- 🐧 **操作系统**: openEuler、银河麒麟、统信UOS等国产OS

## 💻 使用方法

### 方法1: Python API

```python
from deployment_predictor_xinchuan import DeploymentResourcePredictorXinChuan

# 创建预测器(选择信创模式)
predictor = DeploymentResourcePredictorXinChuan(xinchuan_mode='standard')
# 可选模式: 'off'(关闭), 'standard'(标准), 'strict'(严格), 'full'(完全)

# 输入需求
input_data = {
    'data_size_gb': 5000,           # 数据量5TB
    'transactions_per_day': 5000000, # 每天500万笔交易
    'max_connections': 2000,         # 最大连接数
    'business_type': 'OLTP',        # 业务类型
    'high_availability': True,       # 高可用
    'disaster_recovery': False       # 容灾
}

# 生成预测
result = predictor.predict(input_data)

# 查看信创信息
print("信创模式:", result['xinchuan_info']['mode'])
print("服务器品牌:", result['xinchuan_info']['servers'])
print("成本优势:", result['xinchuan_info']['cost_advantage'])

# 查看设备清单
for item in result['equipment_list']:
    print(f"{item['name']} - {item['vendor']} - ¥{item['total_price']:,}")

# 查看成本对比
if result['cost_breakdown']['xinchuan_comparison']:
    comp = result['cost_breakdown']['xinchuan_comparison']
    print(f"信创方案: ¥{comp['xinchuan_cost']:,.0f}")
    print(f"国外品牌: ¥{comp['international_cost']:,.0f}")
    print(f"节约: ¥{comp['cost_savings']:,.0f} ({comp['savings_percent']}%)")
```

    data = request.json
