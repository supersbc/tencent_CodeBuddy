🚀 使用方法

【Web界面】（推荐）

1. 访问模型库管理
   http://localhost:5003/model_library

2. 创建自定义库
   - 点击"➕ 创建自定义模型库"
   - 填写库信息
   - 点击"创建"

3. 添加案例
   - 通过预测结果保存
   - 或使用Python脚本添加

4. 导出分享
   - 点击"导出"按钮
   - 下载JSON文件
   - 分享给团队

5. 导入使用
   - 点击"📥 导入模型库"
   - 选择JSON文件
   - 点击"导入"

【Python脚本】

```python
from custom_model_builder import CustomModelBuilder

builder = CustomModelBuilder()

# 创建库
result = builder.create_custom_library(
    name="我的模型库",
    description="基于实际项目经验",
    industry="金融"
)

# 添加案例
builder.add_case_to_library(
    result['library_id'],
    input_data={...},
    output_data={...}
)

# 导出
builder.export_library(
    result['library_id'],
    'my_library.json'
)
```
