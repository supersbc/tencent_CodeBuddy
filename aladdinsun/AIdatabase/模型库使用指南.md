## 🚀 使用方法

### 方法1: Web界面（推荐）

1. **访问模型库管理页面**
   ```
   http://localhost:5003/model_library
   ```

2. **浏览可用模型库**
   - 查看每个模型库的详细信息
   - 对比案例数、准确率、行业覆盖

3. **下载模型库**
   - 点击"下载安装"按钮
   - 等待下载完成（几秒钟）
   - 系统自动保存到本地

4. **使用模型库**
   - 点击"使用此模型库"
   - 系统自动加载案例到训练集
   - 开始使用新模型进行预测

5. **管理模型库**
   - 查看已安装的模型库
   - 删除不需要的模型库
   - 切换不同的模型库

### 方法2: Python脚本

```python
from model_library_manager import ModelLibraryManager

# 初始化管理器
manager = ModelLibraryManager()

# 列出所有可用模型库
libraries = manager.list_available_libraries()
for lib in libraries:
    print(f"{lib['name']}: {lib['cases']}个案例, {lib['accuracy']}")

# 下载模型库
result = manager.download_library('tencent_cloud_official')
print(f"下载完成: {result['cases']}个案例")

# 加载模型库
cases = manager.load_library('tencent_cloud_official')
print(f"加载了{len(cases)}个案例")
```

### 方法3: 命令行

```bash
# 查看可用模型库
python3 model_library_manager.py

# 下载推荐的模型库
python3 -c "from model_library_manager import ModelLibraryManager; \
m = ModelLibraryManager(); \
m.download_library('tencent_cloud_official')"
```

    'community_finance',        # 行业专用库

for lib_id in libraries:
    manager.download_library(lib_id)
    cases = manager.load_library(lib_id)
    # 加载到训练系统...

## 🔧 故障排除

### 问题1: 下载失败
**解决方案**:
- 检查网络连接
- 重试下载
- 使用备用源

### 问题2: 加载失败
**解决方案**:
- 检查文件完整性
- 重新下载模型库
- 查看错误日志

### 问题3: 准确率不理想
**解决方案**:
- 尝试其他模型库
- 组合多个模型库
- 添加自定义案例

### 问题4: 存储空间不足
**解决方案**:
- 删除不用的模型库
- 只保留必要的库
- 清理临时文件
