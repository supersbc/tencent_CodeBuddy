# 模型库快速使用指南

## 🚀 5分钟快速上手

### 步骤1: 启动系统
```bash
cd /Users/aladdin/Documents/gitdata/tencent_CodeBuddy/aladdinsun/AIdatabase
python3 app_simple.py
```

### 步骤2: 访问模型库管理页面
打开浏览器访问: `http://127.0.0.1:5173/model_library`

### 步骤3: 浏览预置模型库
系统已为您准备了8个高质量预置模型库：

1. ⭐ **腾讯云官方模型库** - 通用场景，推荐新手
2. 💰 **金融行业社区模型库** - 金融行业专用
3. 🛒 **电商行业社区模型库** - 电商平台优化
4. 🎮 **游戏行业社区模型库** - 游戏低延迟
5. 🌐 **GitHub开源模型库** - 免费开源
6. 🤖 **HuggingFace模型库** - AI/ML优化
7. 🏆 **Kaggle竞赛模型库** - 极致性能
8. ☁️ **阿里云模型库** - 双11实战

### 步骤4: 激活模型库
点击您需要的模型库，然后点击"激活"按钮

### 步骤5: 开始预测
返回"部署资源预测"页面，输入参数，系统将使用激活的模型库进行智能预测

## 📚 详细使用教程

### 场景1: 金融行业用户

**需求**: 银行核心系统，高可用要求

**推荐模型库**: 金融行业社区模型库

**操作步骤**:
1. 访问模型库管理页面
2. 找到"金融行业社区模型库"
3. 点击"查看详情"，了解案例
4. 点击"激活"按钮
5. 返回预测页面，输入参数：
   - 数据规模: 500GB
   - QPS: 20000
   - 高可用: 是
   - 灾备: 是

**预期结果**:
- 架构类型: 分布式

- 准确率: 93-96%
- 包含灾备方案

## ❓ 常见问题

### Q1: 如何知道哪个模型库最适合我？
**A**: 参考"模型库选择决策树"，或查看"预置模型库说明.md"

### Q2: 可以同时激活多个模型库吗？
**A**: 不可以，系统一次只能使用一个模型库进行预测

### Q3: 预置模型库可以修改吗？
**A**: 预置模型库是只读的，但您可以基于它创建自定义模型库

### Q4: 如何查看模型库的详细案例？
**A**: 在模型库管理页面点击模型库名称，查看所有案例

### Q5: 模型库会自动更新吗？
**A**: 预置模型库会定期更新，您可以重新下载获取最新版本

### Q6: 如何评估预测准确率？
**A**: 每个模型库都有准确率指标，您也可以通过实际使用验证

### Q7: 可以贡献案例到模型库吗？
**A**: 可以！使用"自主训练"功能提交您的案例

### Q8: 模型库占用多少存储空间？
**A**: 8个预置模型库总共约70KB，非常轻量

## 🔧 故障排除

### 问题1: 模型库加载失败
**解决方案**:
1. 检查文件是否存在于 `model_libraries/` 目录
2. 运行验证脚本: `python3 verify_model_libraries.py`
3. 重新启动服务

### 问题2: 预测结果不准确
**解决方案**:
1. 尝试切换到其他模型库
2. 检查输入参数是否合理
3. 查看相似案例作为参考

### 问题3: 找不到合适的模型库
**解决方案**:
1. 使用"腾讯云官方"通用模型库
2. 或创建自定义模型库
3. 提交需求到社区
