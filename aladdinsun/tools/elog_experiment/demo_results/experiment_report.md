# E-Log 实验报告

**实验时间**: 2025-11-02 19:20:37

**实验目标**: 测试不同触发阈值对日志体积、吞吐量、准确率的影响

## 1. 实验配置

- **数据库**: Apache IoTDB v1.2.2
- **基准测试**: TSBS (Time Series Benchmark Suite)
- **测试阈值**: θ = [0.00, 0.01, 0.02, 0.03, 0.05, 0.10]
- **固定参数**: α = 100 (准确率权重)
- **计算公式**: θ = β / (α + β)

## 3. 结论与建议

基于实验结果，推荐配置：

- **推荐θ值**: 0.01 (论文推荐值)
- **对应α**: 100
- **对应β**: 1
- **理由**: 在保持高准确率的同时，最小化日志开销和性能影响
